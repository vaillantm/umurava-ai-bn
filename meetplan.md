# Hackathon Project Plan: AI-Powered Screening ATS

## 1. Project Overview
Goal: Build an AI-powered, self-service ATS that lets recruiters screen multiple resumes at once using programmable rubrics.

Target user: HR professionals and TA managers.

## 2. Core Features
- Job creation and applicant upload
- Resume parsing with OCR-capable handling for scanned PDFs/images where possible
- AI shortlisting with deep reasoning and batch analytics
- Incomplete resumes separated for manual review
- Login wall for recruiter access

## 3. Technical Requirements
- Google Gemini API is mandatory
- Strict AI response validation
- Product-agnostic deployment

## 4. AI Strategy
- Batch processing for larger candidate sets
- Transparent reasoning and fairness-oriented output
- Mock data support for demo runs

## 5. Demo Flow
1. Log in
2. Create a job
3. Upload or create 20-30 candidates
4. Run screening
5. Review shortlist, incomplete candidates, and exportable output
